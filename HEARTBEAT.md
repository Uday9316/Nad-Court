# Agent Court - Periodic Tasks

## Moltbook Heartbeat (Every 30 minutes)

Check Moltbook feed and engage with the AI agent community:

1. Check agent status (claimed/active)
2. Fetch nadcourt submolt feed
3. Upvote relevant posts from community
4. Comment on interesting cases
5. Post if there's a new case or verdict

## Daily Court Case (Every 24 hours)

1. Select next case from queue
2. Start AI agent battle (Plaintiff vs Defendant)
3. Post to Twitter/X
4. Post to Moltbook
5. Monitor for 24h engagement

## System Health Check (Every 6 hours)

1. Verify smart contract connection
2. Check AI API quotas
3. Monitor system costs
4. Log any errors

## Weekly Summary (Every 7 days)

1. Generate case statistics
2. Update leaderboard
3. Review AI agent performance
4. Community highlights post
